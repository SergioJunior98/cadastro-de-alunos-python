# 🎓 Sistema de Cadastro de Alunos

Sistema desenvolvido em Python para gerenciamento de alunos, utilizando banco de dados SQLite e estruturação com funções.

---

## 🚀 Funcionalidades

* ✅ Cadastro de novos alunos
* 📋 Listagem de alunos cadastrados
* ❌ Remoção de aluno por ID
* 🔍 Validação de idade com tratamento de erro (`try/except`)
* ✍️ Formatação automática do nome (`capitalize`)
* 📌 Menu interativo com `match case`

---

## 🛠️ Tecnologias utilizadas

* Python
* SQLite
* Programação estruturada com funções (`def`)

---

## ▶️ Como executar

1. Instale o Python (versão 3.10 ou superior)
2. Clone este repositório:

```
git clone https://github.com/seu-usuario/registro-de-alunos.git
```

3. Acesse a pasta do projeto:

```
cd registro-de-alunos
```

4. Execute o programa:

```
python main.py
```
